  • Fecha de publicación: 9 Diciembre 2024
  • Vea Novedades de Thymeleaf 3.1 y cómo migrar.

Maven/Gradle

La forma más sencilla de incluir Thymeleaf en tu proyecto es usar un sistema de compilación como Maven o Gradle y aprovechar los artefactos de Thymeleaf que se encuentran en el Repositorio Central. Solo necesitas agregar las dependencias de Thymeleaf que necesitas a tu proyecto:

Módulo ID de Grupo ID del Artefacto
Librería principal org.thymeleaf thymeleaf
Integración con Spring 5 org.thymeleaf thymeleaf-spring5
Integración con Spring 6 org.thymeleaf thymeleaf-spring6
Integración con Spring Security 5 org.thymeleaf.extras thymeleaf-extras-springsecurity5
Integración con Spring Security 6 org.thymeleaf.extras thymeleaf-extras-springsecurity6
Librería para pruebas org.thymeleaf.testing thymeleaf-testing
Librería para pruebas - Integración con Spring 5 org.thymeleaf.testing thymeleaf-testing-spring5
Librería para pruebas - Integración con Spring 6 org.thymeleaf.testing thymeleaf-testing-spring6

Paquetes de distribución

En caso de que no utilice una herramienta de compilación como Maven o Gradle, también ofrecemos un paquete de distribución completo que contiene binarios, código fuente, javadoc y dependencias en forma de un conveniente archivo .zip, que puede descargar desde nuestra distribución de lanzamientos de GitHub.

Fuentes

El código fuente de Thymeleaf está disponible en GitHub en https://github.com/thymeleaf/thymeleaf
