Thymeleaf

El ecosistema de Thymeleaf se compone de herramientas y extensiones de dos tipos:

  • Extensiones oficiales (t.c.c. Extras de Thymeleaf): creadas por el Equipo de Thymeleaf y mantenidas como parte del proyecto.
  • No oficiales o Extensiones de la Comunidad: vinculadas desde esta página pero desarrolladas y mantenidas por usuarios de Thymeleaf fuera del Proyecto Thymeleaf y distribuidas bajo sus propias licencias y términos de soporte.

Herramientas Oficiales y Extensiones

Biblioteca de pruebas de Thymeleaf

https://github.com/thymeleaf/thymeleaf-testing

La biblioteca de pruebas de Thymeleaf permite a los desarrolladores crear pruebas automatizadas para aplicaciones y extensiones de Thymeleaf de forma sencilla y declarativa. Entre sus características se encuentran:

  • Funciona como una biblioteca independiente, invocable desde múltiples marcos de prueba como, por ejemplo, JUnit.
  • Prueba sólo la capa de vista: procesamiento de la plantilla y su resultado.
  • Permite especificar la entrada de prueba como fragmentos de marcado y también como enlaces a los archivos de plantilla de una aplicación basada en Thymeleaf que se está desarrollando, para probar fácilmente su capa de vista.
  • Artefactos de pruebas versátiles: secuencias de pruebas, iteración, ejecución concurrente...
  • Integración de Spring Framework y Spring Security.

Complemento para Eclipse de Thymeleaf

https://github.com/thymeleaf/thymeleaf-extras-eclipse-plugin

El complemento Thymeleaf para Eclipse IDE agrega funciones de asistencia de contenido que hacen que trabajar en plantillas de Thymeleaf sea más agradable y mucho más cómodo.

  • Completado de código de los procesadores de atributos Thymeleaf.
  • Explicaciones detalladas para cada atributo.
  • Asistencia de contenido dentro de expresiones: objetos de utilidad de expresión.

Thymeleaf + Apache Tiles 2

https://github.com/thymeleaf/thymeleaf-extras-tiles2

La biblioteca Thymeleaf Extras Tiles 2 proporciona un dialecto que permite usar Apache Tiles 2 como método de diseño para aplicaciones basadas en Thymeleaf. Características:

  • Utilice plantillas Thymeleaf en las definiciones de Tiles.
  • Utilice fragmentos de plantillas de Thymeleaf en las definiciones de diseño, ¡plantillas naturales posibles con Tiles!
  • Mezcle plantillas JSP y Thymeleaf en el mismo diseño si es necesario.
  • Integraciones opcionales de Spring MVC 3 y Spring Web Flow 2.3.

Thymeleaf + Spring Security

https://github.com/thymeleaf/thymeleaf-extras-springsecurity

La biblioteca Thymeleaf Extras de Spring Security proporciona un dialecto que permite integrar diversos aspectos de autorización y autenticación de Spring Security (versiones 3.x, 4.x y 5.x) en aplicaciones basadas en Thymeleaf. Características:

  • Equivalente basado en Thymeleaf a la biblioteca de etiquetas JSP de Spring Security .
  • Agrega nuevos objetos de utilidad de expresión como #authentication y #authorization para integrar las capacidades de Spring Security en las expresiones de Thymeleaf.
  • Agrega nuevos atributos como sec:authentication y sec:authorized para una configuración más sencilla de la seguridad.

https://github.com/thymeleaf/thymeleaf-extras-conditionalcomments

La biblioteca de comentarios condicionales de Thymeleaf Extras proporciona soporte para los comentarios condicionales de Internet Explorer en las plantillas de Thymeleaf.

Dialectos comunitarios

Desarrollado y mantenido por usuarios de Thymeleaf fuera del Proyecto Thymeleaf y distribuido bajo su propia licencia y términos de soporte.

Dialecto de diseño de Thymeleaf

por Emanuel Rabina

https://github.com/ultraq/thymeleaf-layout-dialect

Un dialecto para Thymeleaf que te permite usar plantillas de diseño/decoración para dar estilo a tu contenido, así como pasar elementos de fragmentos completos a las páginas incluidas, todo para mejorar la reutilización del código. Si alguna vez has usado SiteMesh para tus JSP o Facelets con JSF, los conceptos de esta biblioteca te resultarán familiares.

Proyecto Dandelion (N.d.T: Diente de León, en español)

Por Thibault Duchateau y Romain Lespinasse

https://dandelion.github.io

Dandelion es un framework Java gratuito y de código abierto cuyo objetivo es simplificar el desarrollo web en entornos basados en JVM.

Ayuda a gestionar los recursos web (JS, CSS) de diversas maneras: permite una buena organización y optimizaciones clásicas como la minimización, la fusión, la compresión o el almacenamiento en caché. También facilita la integración de potentes bibliotecas de JavaScript gracias a un conjunto de componentes extensibles.

PagesDialect

Por José Samper

http://pagesdialect.sourceforge.net/

PagesDialect es un dialecto personalizado que proporciona algunas utilidades de procesamiento de colecciones, como paginación, clasificación o exportación.

Thymeleaf Con Dialect

Por Martin Frey

https://github.com/Antibrumm/thymeleaf-extras-with-dialect

Un dialecto para Thymeleaf que permite usar atributos con el prefijo "with" para evitar expresiones th:with largas.

Thymeleaf Cache Dialect

Por Martin Frey

https://github.com/Antibrumm/thymeleaf-extras-cache-dialect

Algunas partes de nuestra página web no cambiarán con frecuencia durante la vida útil de la aplicación o solo dependerán de una sesión de usuario.

Este dialecto almacenará en caché la salida resultante del elemento en el que se declara y lo reemplazará si se produce un acceso a la caché.

HTML5 Validation Dialect

Por Francisco Pérez and José Samper

http://html5valdialect.sourceforge.net/

Este dialecto lee las anotaciones JSR-303 y modifica el código HTML introduciendo código de validación de formulario HTML5 que coincide con las anotaciones.

WUIC Dialect

Por Guillaume Drouet

https://wuic.github.io/

WUIC es un proyecto Java que ayuda a los desarrolladores a gestionar sus estadísticas web. Minificación y agregación de JS/CSS, cálculo de sprites, caché de servidor/navegador con eliminación ante cualquier cambio, etc. El dialecto ayuda a los usuarios de Thymeleaf a utilizar las funciones de WUIC al crear sus plantillas.

Data Attribute Dialect

Por Max Bruchmann

https://github.com/mxab/thymeleaf-extras-data-attribute

El dialecto de atributos de datos de Thymeleaf evalúa las expresiones de Thymeleaf en los atributos data:* y establece el resultado como el atributo data-* de la etiqueta HTML.

Thymeleaf Spring Data Dialect

by Javier Pena

https://github.com/jpenren/thymeleaf-spring-data-dialect

Este dialecto utiliza los elementos de paginación de Spring Data para crear componentes de paginación y clasificación en el estilo Bootstrap.

Thymeleaf Shiro Dialect

Por Thebora Kompanioni

https://github.com/theborakompanioni/thymeleaf-extras-shiro

Un dialecto de Thymeleaf para etiquetas Apache Shiro.

Integraciones de la Comunidad

Integraciones de Thymeleaf para MVC 1.0 (JSR371)

by Tamás Pálfalvi

https://github.com/inbuss/thymeleaf-mvc
https://github.com/inbuss/thymeleaf-cdi
https://github.com/inbuss/thymeleaf-jersey

Una extensión que proporciona Thymeleaf como motor de visualización en entornos JavaEE MVC 1.0 (JSR371). Incluye integración con CDI que expone beans con nombre a expresiones y extiende el servicio de conversión de Thymeleaf mediante la función JAX-RS ParamConverter (solo en Jersey). Estas funciones también están disponibles como extensiones independientes para su uso en entornos que no sean MVC.

HDIV + Thymeleaf

por The HDIV Project

https://www.hdiv.org

HDIV es un marco de trabajo de seguridad de aplicaciones web que ayuda a evitar los riesgos de seguridad más comunes (los diez más importantes de OWASP). HDIV controla el flujo de datos entre el servidor y el cliente, aplicando validaciones de seguridad de forma transparente y sin actualizar el código fuente de las aplicaciones.

HDIV aplica comprobaciones de integridad (evitando, por ejemplo, la manipulación de parámetros y URL) a todos los datos que se envían al cliente desde el servidor, así como validaciones de entrada genéricas a los datos procedentes de los campos de texto. Además, HDIV añade tokens aleatorios a cada enlace y formulario para evitar el riesgo de CSRF en la web.

Módulo de Thymeleaf para Play Framework 1.2

Por Satoshi Takata

https://github.com/choreo/play-thymeleaf

Este módulo permite usar Thymeleaf como sustituto de las plantillas Groovy en la capa de visualización de Play Framework 1.2.

Módulo de Thymeleaf para Play Framework 2 (Scala)

Por Dmitry Avershin

https://github.com/dmitraver/scala-play-thymeleaf-plugin

Un módulo para Play Framework (Scala) que utiliza el motor de plantillas Thymeleaf como alternativa a Twirl, que es un motor de plantillas estándar de Play Framework.

Herramientas de la Comunidad

Bootify.io

https://bootify.io/

Bootify.io es un generador de código para Spring Boot. Se pueden seleccionar tres opciones para el frontend, todas basadas en Thymeleaf, junto con un CSS opcional de Webpack, Bootstrap o Tailwind. El usuario obtiene una aplicación Spring Boot personalizada en la última versión, con la configuración de Thymeleaf, incluyendo plugins, recarga en tiempo real de las plantillas de Thymeleaf durante el desarrollo y un frontend CRUD.

Thymol

Por Jim Benson

https://github.com/thymol/thymol.js

Thymol es una biblioteca de JavaScript útil que permite a los desarrolladores aplicar sus inclusiones de fragmentos th:include/th:replace de forma estática (sin ejecutar la aplicación).

Además, ofrece numerosas funciones potentes para la creación de prototipos estáticos, como la posibilidad de que los desarrolladores evalúen estáticamente fragmentos condicionales (th:if/th:unless) en sus prototipos, lo que mejora la experiencia de desarrollo con plantillas naturales.

thymeleaf-fragment.js

Por Black Pepper Software

https://github.com/BlackPepperSoftware/thymeleaf-fragment.js

thymeleaf-fragment.js proporciona un script sencillo que solo procesa los atributos th:include, th:insert y th:replace mediante jQuery. Esto permite que las plantillas naturales que usan fragmentos se visualicen en el navegador como prototipos estáticos.

Thymus

Por Will Hoover

https://ugate.github.io/thymus

thymus.js es un complemento de JQuery inspirado en Thymeleaf que puede utilizarse como controlador del lado del cliente de Thymeleaf para actualizaciones fragmentadas de modelos/vistas a través del navegador del cliente. thymus.js es lo suficientemente flexible como para ejecutarse independientemente de Thymeleaf, pero también puede utilizarse como un potente complemento de producción para Thymeleaf o como motor de creación de prototipos sin conexión.

Guía de inicio rápido de Spring MVC: Arquetipo Maven

Por Rafał Borowiec

https://github.com/kolorobot/spring-mvc-quickstart-archetype

Un arquetipo de Maven para aplicaciones web, incluyendo Thymeleaf, Spring MVC, Spring Security, Bootstrap, JPA (Hibernate/HSQLDB/Spring Data JPA), MongoDB y JUnit.

Thymeleaf + Arquetipo Maven de Spring MVC

Por Álex Soto

http://www.lordofthejars.com/2012/01/once-upon-time-and-long-ago-i-heard.html

Un arquetipo de Maven para arrancar aplicaciones web usando Spring MVC y Thymeleaf.

Thymesheet

Por Adam Perry

https://connect-group.github.io/thymesheet/

Thymesheet es una extensión para Thymeleaf que permite mover atributos como th:if, th:text, etc., a un archivo CSS3 externo. Esto significa que no se incrusta ninguna lógica Java/de servidor en el HTML, lo que lleva las plantillas naturales un paso más allá. Compatible con Tiles, Spring, comentarios condicionales y cualquier dialecto de Thymeleaf.

Extensión de Thymeleaf para JAWR

Por Danny Trunk

https://github.com/dtrunk90/thymeleaf-jawr-extension

Un dialecto para el motor de plantillas Thymeleaf que permite el uso de Jawr (una solución de empaquetado adaptable para Javascript y CSS que permite el desarrollo rápido de recursos en archivos de módulos separados).

Dialecto SQL y servlet independiente

Por José Samper

https://github.com/jmiguelsamper/thymeleaf-extras-sqldialect

Esta aplicación web de ejemplo presenta una forma de crear páginas web similares a PHP usando Thymeleaf sin necesidad de código Java. Para ello, se introduce un nuevo dialecto SQL y un servlet para la ejecución independiente de plantillas.
