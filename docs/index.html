Thymeleaf

9 Diciembre 2024: Thymeleaf 3.1.3.RELEASE ha sido publicado.
Vea las Novedades de Thymeleaf 3.1 y cómo migrar.

Thymeleaf es un moderno motor de plantillas Java del lado del servidor para entornos web e independientes.

El objetivo principal de Thymeleaf es incorporar plantillas elegantes y naturales a tu flujo de trabajo de desarrollo: HTML que se muestre correctamente en los navegadores y que también funcionen como prototipos estáticos, lo que facilita una colaboración más sólida en los equipos de desarrollo.

Con módulos para Spring Framework, numerosas integraciones con tus herramientas favoritas y la posibilidad de añadir tus propias funciones, Thymeleaf es ideal para el desarrollo web moderno con JVM y HTML5, aunque ofrece muchas más funciones.

Plantillas naturales

Las plantillas HTML escritas en Thymeleaf todavía se ven y funcionan como HTML, lo que permite que las plantillas reales que se ejecutan en su aplicación sigan funcionando como artefactos de diseño útiles.

Integraciones en abundancia

Eclipse, IntelliJ IDEA, Spring, Play e incluso la prometedora API Modelo-Vista-Controlador para Java EE 8. Desarrolla Thymeleaf con tus herramientas favoritas y tu framework de desarrollo web favorito.

Explora nuestro Ecosistema para ver más integraciones, incluyendo plugins desarrollados por la comunidad para acelerar tu desarrollo con Thymeleaf.
