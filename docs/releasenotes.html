Thymeleaf 3.0.15 - Notas de la versión

31 Enero 2022

Thymeleaf 3.0.15 (3.0.15.RELEASE) acaba de publicarse.

Esta es una actualización de seguridad muy recomendable que incluye correcciones de errores y cambios de funciones.

Mejoras de seguridad:

  • Se solucionó la verificación inconsistente de acceso restringido a variables debido al almacenamiento en caché.
  • Detección mejorada de escenarios de ejecución de expresiones restringidas.
  • Detección mejorada de usos restringidos de nombres de vistas en la entrada de solicitud directa.

Esta versión debería funcionar como reemplazo directo de las versiones 3.0.x. Consulta nuestra Página de descargas para saber cómo obtenerla.

Thymeleaf 3.0.12 - Notas de la versión

21 Diciembre 2020

Thymeleaf 3.0.12 (3.0.12.RELEASE) acaba de publicarse.

Esta es una actualización de seguridad muy recomendable que incluye correcciones de errores y cambios de funciones.

Mejoras de seguridad:

  • Se evitó la creación de instancias de nuevos objetos y llamadas a clases estáticas en el modo de evaluación de expresiones restringidas , tanto para escenarios basados ​​en OGNL como en SpringEL.
  • Usuarios de Spring: se evita la ejecución de nombres de vista como expresiones de fragmento cuando el nombre de la vista está contenido en la ruta URL o en los parámetros de consulta.

Problemas solucionados:

  • Se solucionó el problema de que los métodos de utilidad de expresión #numbers.format*(...) no producían números usando los símbolos de dígitos correctos para las configuraciones regionales que los usan (por ejemplo, farsi), en las versiones de JDK donde NumberFormat lo hace.
  • Se solucionó el problema de package-list que no se producía para JavaDoc desde que se comenzó a usar JDK 11 para compilar el proyecto.
  • Usuarios de Spring: se solucionó una pérdida de memoria en ThymeleafViewResolver en las redirecciones a URL creadas dinámicamente.

Cambios de características:

  • Usuarios de Spring 5.x: se agregó el método encode() a los métodos de utilidad de expresión #mvc.url(...).
  • Usuarios de Spring 5.x y Spring WebFlow: soporte adaptado de WebFlow a Spring WebFlow 2.5 después de los cambios en la API (ahora se requiere WebFlow 2.5.0+).

Actualizaciones de dependencia:

  • OGNL actualizado a 3.1.26.
  • Jackson actualizado a 2.11.3.

Esta versión debería funcionar como reemplazo directo de las versiones 3.0.x. Consulta nuestra Página de descargas para saber cómo obtenerla.

Si estás interesado, puedes echar un vistazo a la lista de problemas en GitHub, que suele contener explicaciones más detalladas:

  • thymeleaf: ver hito.
  • thymeleaf-spring: ver hito.

Thymeleaf 3.0.11 - Notas de la versión

29 October 2018

Thymeleaf 3.0.11 (3.0.11.RELEASE) acaba de publicarse.

Esta es una versión de mantenimiento con correcciones menores para un par de problemas introducidos en la versión 3.0.10. Estos problemas afectaron a:

  • Usuarios de JPMS (Sistema de Módulos de la Plataforma Java): algunos módulos de Thymeleaf declararon nombres de módulo no válidos.
  • Usuarios de Spring WebFlux.fn (lado funcional de Spring WebFlux): se generaba una excepción al renderizar plantillas que usaban el dialecto SpringStandard.

Esta versión debería funcionar como reemplazo directo de las versiones 3.0.x. Consulta nuestra Página de descargas para saber cómo obtenerla.

Si actualmente utiliza una versión anterior a la 3.0.10, visite el anuncio de lanzamiento de la versión 3.0.10 para obtener más información sobre las nuevas funciones.
